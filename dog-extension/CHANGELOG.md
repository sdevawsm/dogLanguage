# Changelog

All notable changes to the Dog Language VS Code Extension will be documented in this file.

## [0.1.0] - 2026-05-19

### Added
- ✨ Complete syntax highlighting for Dog Language
- 📝 Comprehensive snippet library (12+ snippets)
- 🔧 Language configuration with auto-indentation
- 📚 Full documentation and examples
- 🎨 Support for block/line comments
- 🚀 Auto-closing brackets and pairs
- 💡 IntelliSense suggestions
- 📖 Theme documentation

### Features
- Syntax highlighting for all Dog keywords (dar_a_pata, focar_no_esquilo, etc)
- Type highlighting (pedigree, pelo, carne, raça, vira_lata)
- Function highlighting (canil, latir, farejar)
- Comment support (// and /* */)
- String highlighting with escape sequences
- Number highlighting (integers and floats)
- Operator highlighting (arithmetic, comparison, logical)

### Snippets
- `canil` - Main function template
- `latir` - Print statement
- `vira_lata` - Generic variable
- `pedigree` - Integer variable
- `dar_a_pata` - If statement
- `dar_a_pata_senao` - If-else statement
- `focar_no_esquilo` - While loop
- `funcao` - Function declaration
- `matilha` - Class declaration
- `trazer_bolinha` - Return statement
- `chamar_matilha` - Import statement
- `programa` - Complete program template

### Configuration
- Auto-indentation for AU/UAU blocks
- Auto-closing pairs for (), [], {}, \"\"
- Block/line comment support
- Bracket matching
