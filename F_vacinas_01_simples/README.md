# F Vacinas 01 - Simples

Descrição:
Este programa registra duas vacinas de um pet em um arquivo e, em seguida, lê e mostra o histórico completo.

Como executar:
1. Abra o terminal na pasta `F_vacinas_01_simples`.
2. Execute:
   ```bash
   python3 dog_translator.py vacinas_simples.dog
   ```

Arquivo fonte:
- `vacinas_simples.dog`
