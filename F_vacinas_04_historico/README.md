# F Vacinas 04 - Histórico

Descrição:
Este programa grava um histórico de vacinas e mostra o conteúdo do arquivo junto com a contagem de linhas.

Como executar:
1. Abra o terminal na pasta `F_vacinas_04_historico`.
2. Execute:
   ```bash
   python3 dog_translator.py vacinas_historico.dog
   ```

Arquivo fonte:
- `vacinas_historico.dog`
