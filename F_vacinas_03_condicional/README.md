# F Vacinas 03 - Condicional

Descrição:
Este programa grava vacinas em arquivo e verifica se o registro existe antes de mostrar as informações.

Como executar:
1. Abra o terminal na pasta `F_vacinas_03_condicional`.
2. Execute:
   ```bash
   python3 dog_translator.py vacinas_condicional.dog
   ```

Arquivo fonte:
- `vacinas_condicional.dog`
