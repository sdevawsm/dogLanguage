# F Vacinas 06 - Pets Múltiplos

Descrição:
Este programa registra vacinas para dois pets diferentes em arquivos distintos e mostra o histórico de cada um.

Como executar:
1. Abra o terminal na pasta `F_vacinas_06_pet_multiplos`.
2. Execute:
   ```bash
   python3 dog_translator.py vacinas_multiplos.dog
   ```

Arquivo fonte:
- `vacinas_multiplos.dog`
