# F Vacinas 02 - Função

Descrição:
Este programa usa funções para gravar vacinas em arquivo e mostrar o conteúdo do arquivo.

Como executar:
1. Abra o terminal na pasta `F_vacinas_02_funcao`.
2. Execute:
   ```bash
   python3 dog_translator.py vacinas_funcao.dog
   ```

Arquivo fonte:
- `vacinas_funcao.dog`
