# F Vacinas 05 - Contagem

Descrição:
Este programa registra vacinas e usa uma função para contar quantas entradas existem no arquivo.

Como executar:
1. Abra o terminal na pasta `F_vacinas_05_contagem`.
2. Execute:
   ```bash
   python3 dog_translator.py vacinas_contagem.dog
   ```

Arquivo fonte:
- `vacinas_contagem.dog`
